# Shinhan-Big-Data-Hackathon(2022 Sep~Oct)
## 대회 소개
빅데이터 혁신공유대학 사업단에서 개최한 "제1회 신.빅.해-(2트랙)데이터분석" 부문에 숙명여대 소속 5명의 팀원(팀명 : 데이런)들과 함께 참가

### 공모주제 소개
#### 과제 설명
- "증권 성향"은 증권사에서 거래를 할 가능성이 높은 고객을 의미하며, 신한카드 거래고객 중 증권성향의 고객을 예측하는 것이 과제 주요내용임
- 증권 성향고객(Y)이라는 정의에 제공된 데이터에서 정해진 정답은 없으며, 제공 된 카드 소비데이터 중 증권성향고객 여부에 대해 참고할 수 있는 변수로는 금투 활동고객(TF), 결제은행(신한은행/기타은행사/기타증권사)변수가 있음
- 위에 제시한 두 변수를 활용하여 증권성향고객을 정의해도 되고, 다른 변수들을 통해 창의적인 증권성향고객을 정의해도 무방함
#### 방향성
제공되는 데이터의 기간이 1개월로 한정되어 있고 익명화 처리로 인해 모형에 대한 예측력을 높이기에 다소 제약이 있으므로 예측 정확도 보다는 모형에 대한 설계, 접근방식, 향후 실제 활용할 수 있는 방안등을 중심으로 구성

데이터를 통해
1. 증권성향고객에 대한 정의를 어떻게 했는지
2. 전체적인 모형에 대한 설계 및 분석에 활용한 변수는 어떻게 했는지
3. 예측한 모형을 근거로 한 증권성향고객의 특징에는 어떤게 있는지
4. 향후 증권성향으로 예측된 고객들을 당사에서 어떻게 활용 또는 접근해볼 수 있는지 중심으로 구성할 것
